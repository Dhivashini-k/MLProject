# Child Malnutrition Analysis System

## Overview
This machine learning project analyzes child malnutrition using UNICEF Joint Child Malnutrition Estimates (JME) data.

## Features
- Data preprocessing
- Regional malnutrition analysis
- Risk score calculation
- Trend visualization
- Child welfare dashboard

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Dataset
UNICEF Joint Child Malnutrition Estimates (JME)
